WEBRTC

WebRTC (Web Real-Time Communication) facilitate peer-to-peer and direct communication between two browsers. It provides web browsers and mobile applications with real-time communication (RTC) via simple application programming interfaces (APIs). It is a very exciting, powerful, and highly disruptive cutting-edge technology and standard.

WHO IS BEHIND WEB RTC ?

WebRTC is an open-source project supported not only by Google but also by Mozilla and Opera. Furthermore, there are lots of other companies, that are working on particular parts of the project. So, for example, Cisco and Ericsson are working on this in standardisation forums. Thus, the API and underlying protocols are being developed at the World Wide Web Consortium (W3C) and Internet Engineering Task Force (IETF).

SUPPORTED BROWSERS AND PLATFORMS

WebRTC enables multi-platform (Microsoft Windows, iOS, Android etc) support via the use of a WebRTC capable browser (Google Chrome, Firefox, Opera)
